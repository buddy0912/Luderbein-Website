# Workbereich – Wandern Trail Startpack

Dieser Ordner ist die Arbeitsflaeche fuer die spaetere Motivproduktion des Packs `wandern-trail-startpack`.

Zweck:
- vorbereitete Zwischenstaende
- manuelle Nachbearbeitung
- stilistische Varianten
- produktionsnahe Arbeitsdateien vor dem finalen Export

Wichtig:
- In diesem Ordner liegen keine aktiv eingebundenen Tool-Assets.
- Die bestaetigten Rohmotive bleiben unter `../source/raw/`.
- Finale freigegebene Produktionsdateien gehoeren spaeter nach `../final/`.
