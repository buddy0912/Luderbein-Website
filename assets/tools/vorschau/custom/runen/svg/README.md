# SVG-Ablage fuer echte Runen

Dieser Ordner enthaelt den aktiven dateibasierten Runenbestand fuer das Vorschau-Tool.

Regeln:
- pro Datei genau ein Zeichen
- nur echte SVG-Dateien
- keine PNG-, JPEG- oder WebP-Dateien in der aktiven Symbolquelle
- keine automatisch aus Vorlagen geratenen Zeichen
- keine umgedeuteten Google-Symbole
