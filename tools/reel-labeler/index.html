Was bedeuten die Felder?
• Kategorie (cat): interne Sortierung (Holz/Glas/…); wird fürs Filtern genutzt.
• Tag: grobes Label (Material/Person/Projekt/QA ODER frei). Ebenfalls fürs Filtern.
• Freitext: dein Textbaukasten (z.B. „Fichte 18mm“, „Schlüsselanhänger“). Optional.
• Caption (cap): Text, den der Kunde in der Kachel/Preview sieht (kurz & hübsch).
• Alt-Text (alt): Beschreibung fürs Bild (SEO/Barrierefreiheit). Darf ausführlicher sein.
Tipp: Cap ohne Kategorie ist okay. Kategorie bleibt intern im Feld cat.

Label

Kategorie (cat)
Interne Kategorie für Filter/Category-Link (cat). Kunden sehen nicht direkt die Kategorie.
Tag
Interner Gruppen-Tag (z. B. Material, Person, Projekt, QA). Kunden sehen den Tag nicht direkt.
Tag (frei)
Eigener Tag überschreibt die Tag-Buttons.
Freitext für Caption (optional)
Schnell-Input: befüllt Caption (und ggf. Alt-Text), wenn aktiviert.
Caption (cap)
Sichtbarer Text im Reel (Kunde sieht das).
Alt-Text (alt)
Accessibility/Tooltip. Meist nur bei Bildfehlern/Screenreadern sichtbar; optional mit cat • cap.
Workflow: 1) X eingeben 2) Platzhalter anlegen 3) labeln 4) JSON herunterladen 5) in GitHub Datei komplett ersetzen
